# Toni Kroos — Post #1

### _"The Metronome Nobody Measured."_

Standard passing stats — volume, completion % — file Toni Kroos with the
pack: a tidy, high-percentage passer among many. This first post argues the
pack chart is measuring the wrong thing. Change the instrument — rhythm,
geography, risk-free progression, centrality — and the same player turns into
the lone dot, off the scale, in whichever direction you point it.

The honest spine of the analysis is that the outlier **arrives over time**.
At Bayern in 2013/14 and the 2014 World Cup, Kroos is elite but not singular —
a top-three progressor and metronome, sharing the frame with Schweinsteiger,
Lahm, Gago, Mascherano. By Euro 2024 he is alone on every instrument: most
ball receipts resolved per 90, ~3× the next player's betweenness in Germany's
pass network, a tempo and switch profile no one else in the tournament
touches. The post follows that arc — **Doubted (2014) → Trusted (Madrid) →
Alone (2024)** — across four tournaments, the full 2015/16 La Liga season,
three Champions League finals, and his last Bayern league season, all
converted to one canonical 105×68 m schema so every metric compares cleanly.

Every claim in the post cites a figure generated by code in this repo.

## Reproduce the figures

Run from the repo root (regenerates all 23 PNGs from cached data):

```bash
.venv/Scripts/python -X utf8 players/toni-kroos/run_all.py
```

## What's here

- `notebooks/` — 16 numbered scripts that build the figures, plus
  `_tournaments.py`, a shared four-tournament per-match builder.
- `figures/` — the 23 publication PNGs.
- `source/` — reference data (transfer fees; the YouTube transcript is
  gitignored).
- `blog/` — the written essay and its Word export, **kept private** (not on
  GitHub); `build_docx.py` renders it with the key figures embedded.
